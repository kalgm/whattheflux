# React / Redux Boilerplate

Plug and play setup for React and Redux with examples.

---

## Libraries
- React
- Reduct
- Material-UI
- LoDash

## Requirements

- [Node](https://nodejs.org) 4.0 or newer

## Installation

Clone repo and run:
After cloning the repository, install dependencies:

```
npm install
```

## Starting App

```
npm start
```

Open `http://localhost:3000`
